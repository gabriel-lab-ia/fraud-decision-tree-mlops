# Diagrams

This directory contains static architecture diagrams that render directly on GitHub
and in MkDocs.

| File | Purpose |
|---|---|
| `fraud-mlops-big-tech-architecture.svg` | Main layered architecture diagram used by the README and architecture docs. |
| `fraud-mlops-underground-graph.svg` | Compact system graph used as a visual overview. |
| `fraud-mlops-architecture.mmd` | Mermaid source for the simplified architecture flow. |

The SVG files are generated from scripts in `scripts/docs/`. Generated site output
belongs in `site/` and is ignored by Git.
