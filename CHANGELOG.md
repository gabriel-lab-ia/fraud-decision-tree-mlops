# Changelog

## 0.3.0 - Production Hardening, Governance, and Observability

- Added governed model artifact schema and SHA-256 manifest validation.
- Added data contract validation, deterministic dataset fingerprints, and
  train/validation/test splitting.
- Added threshold selection, calibration support, richer metrics, baseline
  comparison, and stricter model validation gates.
- Added versioned API endpoints, batch prediction, request IDs, readiness,
  version metadata, and Prometheus-compatible metadata metrics.
- Added production security posture validation, optional API-key authentication,
  CORS controls, Docker hardening, and Bandit gate.
- Added PSI drift helper, data-quality reports, reference profiles, runbook,
  threat model, Responsible AI, and deployment documentation.

## 0.2.0 - Observability Demo

- Added API demo workflow, telemetry smoke checks, monitoring summary, and model
  explainability export.
